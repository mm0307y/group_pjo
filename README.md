# 여행 코스 추천해주는 서비스
## 1. 목표와 기능
### 1-1 목표
- Spring Boot와 React 를 사용해서 웹 개발 공부를 한다.
- 내가 가고 싶은 나라와 날짜를 선택하면 AI가 코스를 추천해 준다.
- 프론트 : DB연동 하기
- 백앤드 : 다양한 API를 구현하고 AI를 사용해보기

### 1-2 기능
- 여행 코스 짜주기
- 리뷰 및 게시판 구현
- 각 나라에 문화(축제)등을 추천
